## Instructions

1. Git clone the repo
```
$ git clone git@github.com:jbh0630/Car_Ownership_API.git
```
2. Install dependencies
```
$ bundle install
```
3. Create Database
```
$ rails db:create
```
4. Migrate Database
```
$ rails db:migrate
```
5. Run Rails Server
```
$ rails s
```
6. API will run on http://localhost:3000
7.  Routes on http://localhost:3000/api/v1/
